# fireTR terminal

This application created with Python 3.10

## List of available commands:

- **ls** - show all files in current directory
- **mkdir** - create directory name
- **rmdir** - remove directory name
- **cat** - read file in directory name
- **crf** - create file in directory name
- **version** - current version of CommandXS

## How to launch:
1. create a folder for the terminal
2. create python file in this folder
3. copy code to python file
4. launch with powershell
5. you can use it
